# Nuestro primer año

Web romántica y cinematográfica hecha con React, TypeScript, Vite, CSS moderno y Framer Motion.

## Instalación

```bash
npm install
```

## Ejecutar en desarrollo

```bash
npm run dev
```

## Fotografías

Coloca o sustituye las doce fotos en:

```text
src/assets/photos/
```

Los nombres esperados son:

```text
julio.jpg
agosto.jpg
septiembre.jpg
octubre.jpg
noviembre.jpg
diciembre.jpg
enero.jpg
febrero.jpg
marzo.jpg
abril.jpg
mayo.jpg
junio.jpg
```

## Cambiar frases y notas

Edita:

```text
src/data/memories.ts
```

Ahí puedes cambiar el mes, la frase frontal, la nota del reverso, la rotación y la posición aproximada de cada Polaroid.

## Música

La música es opcional. Para activarla, coloca este archivo:

```text
src/assets/music/song.mp3
```

Si no existe, la web sigue funcionando y el botón mostrará que no hay canción disponible.

## Producción

```bash
npm run build
```

Para previsualizar el resultado compilado:

```bash
npm run preview
```
